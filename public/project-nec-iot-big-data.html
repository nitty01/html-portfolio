Technical Architecture

  • Data Flow: Raw data ingestion from IoT sensors → MMS processing → Big data stack transformation → User-accessible analytics dashboard
  • Components: NEC MMS (Micro Modular Server), Hadoop ecosystem, Apache Spark, Kafka streaming, MongoDB/PostgreSQL databases
  • Integration: MMS serves as the core platform hosting all big data services, with seamless integration between data ingestion, processing, and storage layers
  • Scalability: MMS modular architecture allows horizontal scaling of big data components based on workload demands and user requirements
  • Monitoring: Built-in MMS monitoring capabilities combined with big data stack observability tools for system health and performance tracking
